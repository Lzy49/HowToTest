# JavaScript debug terminal 
1. 打开终端 
2. 在终端小箭头中选择 'JavaScript 调试终端(中文)' &  'JavaScript debug terminal'
3. 开始愉快的调试
# Launch.json
1. 点击 debug 窗口
2. 点击创建 launch 
3. 拷贝 调试 配制 https://cn.vitest.dev/guide/debugging.html#vscode
4. 打开要调试的 test , 点击绿色小箭头
# vitest 插件
1. 安装 vitest 插件
2. 开始愉快的调试
## 快键键
搜索 Test:Debug , Test:run 即可找到
